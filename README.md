# リノベーションLP - ペルソナB（30代女性・独身/共働き）向け

## 概要
30代独身女性をターゲットにした、中古リノベーションのランディングページです。
「自分らしさを取り戻したい」という深層心理に訴求する設計になっています。

## ペルソナ設定
- **年齢**: 30代女性
- **状況**: 独身/共働き
- **悩み**: 狭くて機能的でない部屋、インテリアに妥協したくない
- **欲求**: 自己表現・承認欲求、特別な空間での達成感

## デザインコンセプト
- **テーマカラー**: ホワイト × くすみグリーン × ベージュピンク
- **雰囲気**: スタイリッシュ・余白・洗練・"自分時間"
- **写真テイスト**: カフェ・北欧・ナチュラルモダン

## CTA戦略（コンバージョン最適化）

このLPでは、ユーザーの心理状態に合わせて**7箇所にCTA**を配置しています：

1. **ファーストビュー**: 第一印象で興味を持った人向け
2. **共感セクション後**: 痛みに共感した直後に行動を促す
3. **解決策セクション後**: 「これなら解決できそう」と感じた瞬間を逃さない
4. **事例セクション後**: 具体例を見て「自分もできる」と思った人向け
5. **リノベ不動産セクション後**: 物件探しから始めたい人向け
6. **お客様の声後**: 実際の声で背中を押されたタイミング
7. **FAQ後**: 不安が解消された直後に誘導
8. **固定CTAボタン**: いつでもアクセス可能（スクロール連動）

各CTAは文言を変えて、そのセクションの内容に即した誘導を行っています。

## カスタマイズのポイント

### 会社情報の更新
- フッターの会社名、住所、電話番号を実際の情報に変更
- プライバシーポリシーのリンク先を設定

### フォーム送信先の設定
- `js/script.js` の `contactForm.addEventListener` 内で
- 実際のフォーム送信処理（メール送信、データベース保存等）を実装

### 画像の差し替え
- 現在はUnsplashのURLを使用
- 実際のプロジェクト写真に差し替えることを推奨
- `images/` フォルダに画像を配置し、HTMLのsrc属性を更新

### 色のカスタマイズ
- `css/style.css` の `:root` セクションで色を変更可能
  - `--color-white`: #FFFFFF
  - `--color-green`: #A8B5A0
  - `--color-pink`: #E8D5C4
  - `--color-dark`: #3A3A3A
